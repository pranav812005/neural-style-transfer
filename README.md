# 🎨 AI Style Transfer Web App

An AI-based project where I implemented artistic style transfer using deep learning techniques.

The goal of this project was to understand how neural networks can transform images into different artistic styles.  
Users can upload any image and apply styles like **Van Gogh, Picasso, Sketch, and Watercolor**.

## 🚀 Features

- 📤 Upload your own image  
- 🎨 Apply multiple artistic styles  
- ⚡ Fast and simple interface  
- 🖼️ Image preview + download option  

---

## 🧠 How it works

This project is based on **Neural Style Transfer**, where:
- Content image + Style image → combined using deep learning
- The model extracts features and blends them to generate a stylized output

## ⚠️ Note

Due to deployment limitations on Streamlit Cloud, the current live version is a **lightweight demo**.

The full TensorFlow-based model runs locally and can be deployed using:
- Hugging Face Spaces  
- Backend APIs (FastAPI / Flask)  
- Cloud platforms (Render / AWS)
